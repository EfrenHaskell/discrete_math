# Truth tables
Write the Truth Table for
```
(A implies not (B and A))
```
You should have a column of four truth values under each  operator. So it will have the structure with four rows and five columns
```
A B (A implies not (B and A)
T T       ?     ?      ?     
T F
F T
F F
```
